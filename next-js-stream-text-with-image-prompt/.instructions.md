# GitHub Copilot Instructions

## Supported Languages
GitHub Copilot suggestions are restricted to the following languages:
- TypeScript
- JavaScript
- SQL

## Excluded Paths
The following files and folders are excluded from Copilot suggestions:
- `node_modules/`
- `dist/`
- `.next/`
- `prisma/migrations/`

## Project Configuration
- **Package Manager**: pnpm
- **Tools**: Git Bash

## Project Overview
This is a Next.js project powered by Claude (Anthropic's AI SDK v2) for building AI-enabled applications.

### Tech Stack
- **Framework**: Next.js 16.0.3
- **AI SDK**: @ai-sdk/anthropic (v2.0.44), @ai-sdk/react (v2.0.93), ai (v5.0.93)
- **Frontend**: React 19.2.0
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript 5
- **Linting**: ESLint 9

### Available Scripts
- `pnpm dev` - Start development server
- `pnpm build` - Build for production
- `pnpm start` - Start production server
- `pnpm lint` - Run ESLint

## Guidelines for Copilot Assistance
When working on this project, focus on:
1. TypeScript/JavaScript code for Next.js components and API routes
2. React components and hooks using the latest React 19 features
3. Integration with Anthropic's AI SDK for chat functionality
4. Tailwind CSS for styling
5. SQL queries if relevant to the project

## Status
Copilot is enabled for this repository.
